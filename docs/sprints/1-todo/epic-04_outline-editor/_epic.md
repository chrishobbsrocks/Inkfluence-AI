# Epic 4: Outline Editor

## Overview

A dedicated editor for managing book structure after the Knowledge Wizard generates the initial outline. Users can drag-and-drop chapters, add/remove sections, and get AI-powered recommendations for structural improvements and content gaps.

## Success Criteria

- [ ] Chapter tree view with drag-and-drop reordering
- [ ] Sub-section management within chapters
- [ ] Add/remove chapters and sections
- [ ] Word count estimates per chapter
- [ ] AI recommendations panel with coverage analysis
- [ ] AI-suggested chapter additions with accept/dismiss
- [ ] Gap detection highlighting missing topics
- [ ] UI matches wireframe exactly
